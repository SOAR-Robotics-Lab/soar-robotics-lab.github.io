# 招生信息

SOARLAB（飞行通用智能课题组）现面向 2027 届相关专业同学，招收对具身智能、机器人及核心算法感兴趣的直博生、普博生和硕士研究生。课题组即将在南京大学智能科学与技术学院运行，由徐浩老师独立指导。

## 研究主线与平台

我们围绕“两个半”硬件平台展开：无人机、人形机器人，以及正在推动的飞行人形机器人。

| 方向                   | 内容                                                           |
| ---------------------- | -------------------------------------------------------------- |
| Robotics Agent         | 面向 Agent 的感知、规划控制、世界模型与仿真等使能技术。        |
| WAM/VLA 机器人基础模型 | WAM/VLA 驱动的机器人基础模型，是当前最急需推进的方向之一。     |
| 经典机器人核心算法     | 分布式感知、SLAM、无人机集群状态估计、强化学习与自主集群系统。 |

除了传统无人机和机器人系统的全栈积累，课题组已有正在迭代的 Agent 框架和正在训练的 WAM 框架，期待新同学加入共同建设。

## 培养理念与资源

以高水平前沿探索和论文为核心，结合学生基础与兴趣确定课题，反对为发论文而灌水。课题主要围绕不低于 IROS/ICRA/CORL/RSS/RAL 水平的机器人研究进行投稿。

培养方式强调高自由度和高要求：每人方向相对独立，时间灵活、不打卡，但需保持合理沟通频率，不能人间消失。每周一次组会，两周一次一对一（线上或线下）。徐浩老师会亲自带学生上手，提供基础代码，或先做一个最小可跑 demo 帮助热启动；卡住时会一起 debug，必要时也会一起处理硬件问题。

资源方面，课题组将尽力保障计算服务器、实验场地和无人机、机器人等实体验证平台。考虑到正在入职，前期免不了一起建设。对合适的同学，可推荐至业内头部合作单位实习。按照学院惯例发放科研补贴，不低于平均水平。

## 对你的期待

1. **思想品德。** 爱国敬业，拥护社会主义核心价值观；首先需要通过南京大学智能科学与技术学院的保研、考研或申博考核。

2. **学术基础。** 数理与编程功底扎实，对多模态大模型、强化学习、机器学习、SLAM 等方向有经验或强烈兴趣。欢迎数学、物理、人工智能、计算机等相关专业同学跨考。

3. **动手与实验精神。** 我们坚持“没有实物实验，就没有真正的机器人学和具身智能”。希望你愿意学习焊接、组装、3D 打印，能操控航模或机器人。有 RoboMaster 等机器人/无人机竞赛经验者优先。

4. **AI 工具与自学能力。** 入组后会安排自学计划，课题组统一购买 AI 工具 token。欢迎能让 AI 承担编程中 90% 以上工作量的同学，但前提是你有合格的编程基础，能纠正模型错误，并理解计算机组成原理、大模型基本架构和 RL 数学原理。

5. **合作与可靠性。** 课题组风气自由，但学业压力不小。希望每位同学友善、团结、乐于合作，并在具身智能/机器人方向做出扎实成果。

## 项目、实习与毕业

课题组承接的项目以有发表潜力的高水平工作为主，兼顾重要纵横向课题，合作方多为具身智能头部企业。项目投入不会超过总学习研究时间的一半，但需要时该顶就要顶，也不鼓励刻意推脱。鼓励高年级学生进入合作企业或经导师同意的业内头部单位实习，实习期间按企业标准发放工资。
